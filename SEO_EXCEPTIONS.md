
## Sitemap Localization Exceptions

The `sitemap.xml` generated by this project includes localized slugs (e.g., `alternativa-*`) in the `<loc>` elements. This is a deliberate exception to the "English-only technical artifacts" rule to support SEO requirements for localized content.

These entries are critical for search engine indexing of language-specific pages and must be preserved. The `hreflang` clusters correctly point to the language variants.

**Exception Details:**
- **File:** `public/sitemap-0.xml` (and dynamically generated sitemaps)
- **Pattern:** URLs containing `alternativa-` or other localized terms in the slug.
- **Reason:** SEO Preservation.
