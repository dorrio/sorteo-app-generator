## 2024-05-23 - [Metadata/Verification] **Hypothesis:** If we include the winner's name in the Open Graph metadata, the Click-Through-Rate (CTR) on shared links will increase because it leverages social proof and personal relevance. **Implementation:** Added `name` query parameter to verification URLs and updated `generateMetadata` in `app/[locale]/verify/page.tsx` to use it for dynamic titles and descriptions. **Outcome:** Shared links now display "🏆 [Name] is the Winner!" instead of generic text.
## 2025-05-27 - [Metadata/SocialCache] **Hypothesis:** Social platforms cache metadata based on `og:url`. If this URL is static (ignoring query params), all shared winner certificates will display the metadata of the *first* scraped winner, leading to irrelevant previews (e.g., Bob sees Alice's name). **Implementation:** Updated `generateMetadata` in `app/[locale]/verify/page.tsx` to include `id` and `name` query parameters in the `og:url` property. **Outcome:** Each winner verification is now treated as a unique URL, ensuring the correct dynamic image and title are displayed for every share.
## 2025-05-28 - [Verification/Celebration] **Hypothesis:** Adding "Confetti" visuals to the verification page for viewers (not just creators) and optimizing the mobile sticky CTA will increase the "Viewer-to-Creator" conversion rate. Also, fixing bold formatting (`**` vs `*`) for WhatsApp improves message readability. **Implementation:** Added `ConfettiEffect` to `verify-content.tsx`, lowered sticky CTA scroll threshold to 150px, and updated translation hooks for WhatsApp compatibility. **Outcome:** Verification pages are now "celebratory moments" rather than static receipts, encouraging screenshots and viral sharing.
## 2025-05-30 - [SocialProof/DateStamp] **Hypothesis:** Adding a visible, verified date stamp to the Open Graph share image increases the perceived authenticity of the giveaway result ("It happened today/recently" vs "Generic Image"), thereby increasing trust and CTR. **Implementation:** Modified `app/[locale]/verify/page.tsx` to extract the timestamp from the verification ID in `generateMetadata` and pass it to `/api/og`. Updated `/api/og` to render this date in high-contrast Gold/Light Gray. **Outcome:** OG Images now feature a prominent date stamp (e.g., "12/25/2025"), creating a "Breaking News" effect for fresh wins.
